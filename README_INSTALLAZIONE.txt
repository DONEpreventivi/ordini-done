ORDINI D.ONE V18.1.12 — ASSOCIAZIONE CATEGORIE BATCH

CORREZIONE PRINCIPALE
Il backend legge GIACENZE_BATCH per posizione certa delle colonne:
A Batch
B Categoria
C Disponibili
D Quantità minima
E Ultimo aggiornamento
F Operatore
G Attivo

Le categorie vengono quindi associate a ogni singolo batch prima di essere inviate al frontend.

INSTALLAZIONE
1. Sostituire integralmente Code.gs in Google Apps Script.
2. Salvare e creare una NUOVA distribuzione della Web App.
3. Sostituire i file frontend/PWA con quelli inclusi nel pacchetto.
4. Chiudere completamente la PWA e riaprirla. Se necessario rimuoverla dalla Home e reinstallarla per eliminare la vecchia cache.
