ORDINI D.ONE V4 — FLUSSO OPERATIVO

STATO ORDINI
BOZZA → PRONTO → CONDIVISO → INVIATO

INSTALLAZIONE FRONTEND
1. Apri il repository GitHub dell'app ordini.
2. Elimina/sostituisci i vecchi file con quelli di questa cartella.
3. Mantieni la cartella icons.
4. Attendi la pubblicazione GitHub Pages.
5. Su iPhone elimina la vecchia icona Home e aggiungi di nuovo l'app se vedi una versione in cache.

AGGIORNAMENTO APPS SCRIPT
1. Apri il progetto Apps Script collegato al Google Sheet.
2. Sostituisci tutto Codice.gs con Codice_Apps_Script_V4.gs.
3. Salva.
4. Esegui setupDatabase una volta.
5. Deployment → Gestisci deployment → Modifica → Nuova versione → Esegui il deployment.

NOVITÀ
- Ordini in sospeso centrali in Home.
- Bozze condivise tra utenti.
- Stato Pronto separato dalla condivisione.
- Un solo pulsante Condividi ordine.
- PDF, WhatsApp, email, copia e condivisione nativa.
- Dopo la condivisione viene chiesto se l'ordine è stato realmente inviato.
- Gli ordini Condivisi ma non Inviati restano evidenziati.
